# PhotoSort

Keyboard-driven photo triage tool built with Electron. Sort through thousands of photos fast.

## Controls

| Key | Action |
|-----|--------|
| `A` / `←` | Mark for deletion & advance |
| `D` / `→` | Keep & advance |
| `Z` | Undo last decision |

## Setup

```bash
npm install
npm start
```

## Build

```bash
npm run build
```

## Supported formats

JPG, JPEG, PNG, GIF, WebP, BMP, TIFF, HEIC, AVIF
